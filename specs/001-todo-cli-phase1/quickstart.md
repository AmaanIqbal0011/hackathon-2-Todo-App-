# Quickstart: Phase I Todo CLI

## Setup
1. Ensure Python 3.13+ is installed.
2. Install `uv` if not already present.
3. Run `uv sync` to set up the environment.

## Running the App
Run the application using:
```bash
uv run python -m todo.main
```

## Available Actions
- **1. Add Task**: Enter a title and description.
- **2. View Tasks**: See all current tasks.
- **3. Update Task**: Modify an existing task by ID.
- **4. Delete Task**: Remove a task by ID.
- **5. Mark Complete**: Toggle status by ID.
- **6. Exit**: Save and quit.
